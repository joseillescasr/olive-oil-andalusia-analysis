# 🫒 Análisis de la Cadena de Valor del Aceite de Oliva Andaluz

## Descripción
Análisis end-to-end de la cadena de valor del aceite de oliva en Andalucía (2013-2023), 
desde la superficie de olivar hasta los precios percibidos por los agricultores.
Proyecto desarrollado como parte de mi portfolio de transición a Data Analytics.

## Fuente de datos
Anuario de Estadística 2024 — Ministerio de Agricultura, Pesca y Alimentación (MAPA)
- Capítulo 7.12: Olivar (superficies, producciones, precios)

## Stack tecnológico
- **Python (Google Colab)**: limpieza y transformación de datos
- **Google BigQuery**: almacenamiento y análisis SQL
- **Looker Studio**: visualización y dashboard
- **GitHub**: documentación y control de versiones

## Pipeline
Excel MAPA → Python/Colab → BigQuery → SQL → Looker Studio

## Hallazgos principales
- Andalucía produce el **68.37%** del aceite de oliva virgen de España (2023)
- El precio del aceite virgen extra subió un **72.5%** en 2023 respecto a 2022
- El rendimiento de extracción se mantiene constante en ~**20%** independientemente del volumen
- Existe una correlación inversa entre producción y precio, con excepciones documentadas
